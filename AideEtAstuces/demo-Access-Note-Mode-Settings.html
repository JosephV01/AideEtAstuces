Démonstration - Accéder aux paramètres du mode Note

• Cette démonstration montre comment accéder aux paramètres d'une bulle Note via le menu contextuel.
• Faites un clic droit sur la poignée ou sur l'étiquette d'une bulle, puis ouvrez Notes pour afficher les options disponibles.
• L'ensemble des options ne sont disponnible qu'une fois l'option Note activée.
